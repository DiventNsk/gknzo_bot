# Системный подход к работе с проектом

## Общая информация
- Проект: gknzo_bot
- Тип: Telegram бот с Web App приложением
- Библиотека для бота: GrammY
- Директория: C:\Users\cordr\gknzo_bot

## Архитектура
- Telegram бот реализован с использованием библиотеки GrammY
- Web App часть интегрирована с ботом через встроенные веб-приложения Telegram
- Для управления переменными окружения используется библиотека dotenv
- Токен бота должен находиться в файле .env и использоваться через dotenv

## Работа с переменными окружения
- Все чувствительные данные (токены, ключи и т.д.) должны храниться в файле .env
- В коде использовать библиотеку dotenv для загрузки переменных окружения
- Файл .env не должен коммититься в репозиторий

## Язык общения
- На все запросы от пользователя отвечать на русском языке

## Безопасность
- Все данные из файла .env могут быть использованы только после запроса от пользователя и при наличии его согласия
- Никогда не отображать содержимое файла .env в чате
- Обеспечивать безопасное взаимодействие с чувствительными данными, чтобы избежать их утечки

## Примечания
Проект представляет собой Telegram бота с интегрированным веб-приложением, которое позволяет пользователям взаимодействовать с более сложным интерфейсом внутри Telegram.
